Section 02 · Behaviour

Use Case Diagram

Two actors operate the system. The Student performs personal study activities; the Host (a Student who initiates a session) drives Study Room collaboration. Shared cases sit between them.

Section 03 · System

High-Level Architecture

Three-tier architecture with a Next.js client, a dual-runtime backend on Railway (Express + FastAPI), and Supabase as the source of truth. AI work routes through OpenRouter, Gemini, and a local Whisper service.

Frontend

Next.js 15

App Router · TypeScript · Tailwind · shadcn/ui · Vercel deployment
API Layer

Express.js

Node · TypeScript · auth, workspaces, AI orchestration · Railway

Python · Whisper tiny · NLTK + TF-IDF · officeparser · Railway
